# restaurant menu data type

## model (database view):

-   restaurant

    -   id: number
    -   name: text
    -   phone_number: text | null
    -   address: text | null
    -   website: text | null
    -   accepted_payment_methods:
        -   cash: bool | null
        -   girocard: bool | null
        -   credit_card: bool | null
    -   opening_hours: text _//Note: opening_hours osm string implemented by opening_hours.rs_

-   menu_item
    -   restaurant_id
    -   name
    -   category // free form text
    -   price
